# Семестровое задание курса МИФИ - Базы данных
## Структура проекта
db1 - транспортные средства
db2 - автомобильные гонки
db3 - бронирование отелей
db4 - структура организации

## Содержание папок
В каждой папке содержатся sql-скрипты. Файлы, начинающиеся с task... (например, task1) содержат скрипт для получения данных из БД согласно заданию с соответствующим номером.
Файлы, называющиеся create.md содержат скрипт для создания БД (сначала создание структуры, затем наполнение)
